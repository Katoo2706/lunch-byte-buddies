# Features Documentation

This folder contains detailed documentation for all features in the lunch tracking application.

## Current Features

- **Individual Orders** - Single person lunch ordering
- **Team Orders** - Group ordering with individual amounts
- **Global Date Selection** - Unified date picker across the app
- **Settlements** - Financial tracking and balance calculations
- **People Management** - User profiles with gender icons
- **Dashboard** - Overview of orders and statistics

## Feature Documentation

Each feature should have its own markdown file with:
- Overview and purpose
- User interface details
- Business logic explanation
- Implementation notes
- Future enhancements

## Adding New Features

When adding new features, create a corresponding documentation file that follows the established pattern.
