# API Documentation

This folder contains API documentation for the lunch tracking application.

## Structure

- `endpoints.md` - API endpoint documentation
- `schemas.md` - Data schema definitions
- `authentication.md` - Authentication and authorization details

## Getting Started

The lunch app uses Supabase as the backend service, providing:
- Real-time database
- User authentication
- Row-level security
- Automatic API generation

For detailed API usage, refer to the individual documentation files in this folder.
