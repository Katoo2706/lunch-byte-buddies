# Tutorial Documentation

This folder contains step-by-step tutorials for using and extending the lunch tracking application.

## Available Tutorials

- `getting-started.md` - How to use the lunch app for the first time
- `creating-orders.md` - Step-by-step guide for creating lunch orders
- `team-management.md` - Managing team orders and settlements
- `customization.md` - How to customize the app for your team
- `development-setup.md` - Setting up the development environment

## Tutorial Structure

Each tutorial should include:
- Clear objectives
- Prerequisites
- Step-by-step instructions
- Screenshots or examples
- Troubleshooting section
- Next steps

## Contributing

When adding new tutorials, ensure they are:
- Beginner-friendly
- Well-structured
- Include practical examples
- Are regularly updated
